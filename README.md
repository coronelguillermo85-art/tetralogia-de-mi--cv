Esta es la **Landing Page oficial** que integra mis cuatro desarrollos principales. No es solo un portal de enlaces; es el cierre de una narrativa de **Innovación** donde demuestro que la comunicación estratégica y el desarrollo técnico pueden jubilar al PDF estático.

## 🎮 El "Level Select" del Candidato

La landing transforma el currículum en un centro de mando interactivo con estética Synthwave. Cada acceso representa un hito técnico y conceptual:

1.  **Job Seeker Pro:** El origen. Superar filtros automáticos lanzando "CV Attacks".
2.  **Reclutator:** La batalla contra la burocracia y el "Dinosaurio del Reclutamiento".
3.  **HiringKong:** Inversión de roles. Un simulador donde el reclutador debe atrapar el talento antes de que escape.
4.  **ATS Kombat Ultra:** El combate final. Un Fighting Game de 16 bits con IA reactiva contra el algoritmo de filtrado.

## 🛠️ Ingeniería de la Interfaz

Para esta pieza central, prioricé una **Experiencia de Usuario (UX)** inmersiva que no dependiera de assets pesados:

* **Fondo Dinámico (Tron Style):** Desarrollado íntegramente con gradientes CSS y animaciones `@keyframes`. Incluye una rejilla de perspectiva móvil que simula un entorno digital retro-futurista.
* **Conversión Estratégica (CTA):** A diferencia de un portfolio común, el flujo de navegación está diseñado para culminar en una "Conversión" directa hacia mi perfil de **LinkedIn**, facilitando el networking inmediato.
* **Arquitectura Responsiva:** Grid adaptable para que la experiencia sea igual de impactante en escritorio o dispositivos móviles.
